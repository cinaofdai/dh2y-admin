##运行方式
>Linux使用方法

   进入项目目录
```php
    //不带参数
    ./think Controller/Action
    
    //带参数：如 param=hello
    ./think Controller/Action/param/hello
```

>windows使用方法

   进入项目目录
```php
    //不带参数
    think.bat Controller/Action
    
    //带参数：如 param=hello
    think.bat Controller/Action/param/hello
```
