## 简介

tp-dh2y-admin-3.2  ThinkPHP(vue.js)vue.js内嵌结合版本


这样的版本主要利用后台开发人员不懂前端项目开发，只用到了数据绑定功能。

## 安装说明

*  git clone https://github.com/cinaofdai/dh2y-admin.git

*  复制里面的tp-cdn-push文件夹到网站跟目录

*  导入DataBase里面的最新sql文件或者在项目根目录
